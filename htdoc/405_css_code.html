CSS セレクター - CSS

category: CSS

要素を指定するセレクターの種類。

もしあなたが携わるプロジェクトがHTMLとCSSを自由に編集できるプロダクトなら、数種類のCSSセレクターを覚えるだけで要素に装飾をつけれない事案はなくなるでしょう。書き方のテンプレを覚えて、セレクターの使える種類を増やしましょう。

詳しくはMDNの解説サイトをご覧ください。

押さえておくべき 基本のセレクター

全ての要素に適応させるセレクター [全称セレクター]

全称セレクターは「全ての要素を称して」に宣言ブロックを適応させます。

*がCSSでは全称セレクターになります。

一度要素の余白をなくしたい時やbox-sizeing プロパティなどを変更する時につかわれます。
　 ブラウザの種類による見た目の差異を統一すrために書く　地ならし的(ファンデーション)なスタイルを書く時に使います。

/* 全ての要素の余白をなしにする */
         * {
            margin:0;
            padding:0;
        }

CSSセレクター　詳細度:(0,0,0,0)

全称セレクターはより詳細に書かれたセレクターで上書きされます。

特定の要素名で適応させるセレクター　[要素セレクター]

要素セレクターは「要素名」を記述して指定した要素全てに適応させます。

/* p要素の文字サイズを14pxにする */
        p {
        font-size:14px;
    }

このCSSを読み込むHTMLファイル内のp要素全てに適応されます。

CSSセレクター　詳細度:(0,0,0,1)

要素セレクターはクラスセレクターなどで上書きされます。

クラス属性の値で要素を指定する　[クラスセレクター]

クラスセレクターはクラス属性の「値」(クラス値)を記述して, クラス値を持った全ての要素に適応させます。

汎用属性であるクラス属性の値はファイル内に同じ要素名があるときに個別に指定したいときに便利です。

クラス属性は「.」ドット（ピリオド）を前に書いてクラス値を書きます。


<div class="box">
    <div class="item">Item 1</div>
    <div class="item">Item 2</div>
</div>
/* クラス属性の値「item」borderの内側の余白幅を1emにする*/
.item {
       padding:1em;
    }

クラスの値「item」を持ったDiv要素のみが余白1emをとります。

クラスの値「box」を持ったDiv要素には適応されません。

複数のクラスを指定する

複数のクラスを持っている要素もCSSで指定できます。クラス名とクラス名を間隔をあけずに記述します。


        <div class="box">
            <div class="item first">Item 1</div>
            <div class="item">Item 2</div>
        </div>
/* クラス属性の「item」と「first」値をもった要素にしか適応されない*/
        .item.first {
               color:red;
            }

CSSセレクター　詳細度:(0,0,1,0)

クラスセレクターは要素セレクターを上書きできますがIdセレクターは上書きできあません。

id属性の値で要素を指定する　[IDセレクター]

IDセレクターはid属性の「値」(id値)を記述して指定した要素に適応させます。

ID属性はクラス属性とちがい、その特性から同じ値をもつ要素をつくれません。同じ値を持つID値が無いようにコードを書きましょう。

HTML

#ハッシュを前つけてID値を入れる


  <div class="flexbox" id="order">
    <div class="item">Item 1</div>
  </div>

/* id値 oederを持った要素の文字色はredになる*/
        #order {
               color:red;
            }

CSSセレクター 詳細度:(0,1,0,0)

IDセレクターで指定したルールセットはclass以下のルールセットを上書きします。！importantルールもしくはスタイル属性で上書きできます。

要素の状態を指定する[疑似グラスセレクター]

擬似クラス - CSS: カスケーディングスタイルシート | MDN

マウスカーソルの乗った状態、ハイパーリンクのついたテキストなど特定の状態を装飾するセレクター

コロン「:」の後ろに対応するキーワードを入れる。

  • a:linkまだ訪問していないリンクに一致します。
  • a:visited 、ユーザーがすでに訪問したリンクを表します。プライバシー上の理由から、このセレクターを使用して変更できるスタイルはとても限定されています。
  • a:hoverユーザーがポインティングデバイスでアイテムを指した場合（例えば、マウスポインターをその項目にかざした場合）に一致します。
  • a:activeアイテムがクリックされるなど、ユーザーによってアクティブになっているときに一致します。
  • input:focus 入力フォームアイテムに設定されている。ユーザーが入力する項目が確認できるようにする。
