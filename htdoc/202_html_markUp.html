thは表のラベルで変化するデータがなんなのかを総称する言葉をいれる。 tdはテーブルデータで情報をいれる。

scopeはデータの読む方向を指定する属性です。 colは列、表の縦方向に関連づけしていきます。 rowは行、表の横方向の関連付けをします。

人間は表を見るだけで表をどの方向でみるか　なんの表なのか　 初めて見た表でもしばらくながめていればわかるとおもいます。

しかし　見る情報以外でこの情報を伝達する場合　または音声だけで伝える場合scope属性 がないとどちらに読んでいいのか解らないことになります。

アクセシビリティの観点では重要な属性です。

colspan属性 左右でドッキング

初期値は1　セル横軸同士でをドッキングさせる　2以上が入るとセル縦軸で2つをドッキングさせる。その分該当する要素を減らさないといけない。 左右でドッキング

colspan2の場合　隣のtrをさせる削除しないといけない。

rowspan属性 上下でドッキング

初期値は1　セル縦軸同士でをドッキングさせる　2以上が入るとセル縦軸で2つをドッキングさせる。その分該当する要素を減らさないといけない。

rowspanは2の場合　下のtdを削除しないといけない。

CSS border-collapseプロパティ

CSS の border-collapse プロパティは、 <table> の中のセルが境界線を共有するか分離するかを設定します。
border-collapse|MDN このページでは テーブルの特殊なborderプロパティの制御について分かりやすい図で解説しています。表の罫線をセル毎に変化させたい場合は参考にしてください。

border-collapse:collapse;/* 共有 */
border-collapse:separate;/* 分かれる */

CSS border-spacingプロパティ

CSS の border-spacing プロパティは、 <table> における隣り合うセルの枠線同士の間隔を定めます。
このプロパティは border-collapse が separate のときのみ適用されます。
