float clearとレイアウト - HTML

category: HTML

フロートは浮動配置をするためのプロパティです。floatでのレイアウトは過去に作られたウェブサイトでとてもよくつかわれているレイアウトの方法です。過去のサイトのメンテナンスなどをするときに知識不足で挑むとレイアウトがもどらなかったりしますのでポイントを覚えておきましょう。

floatの役割

大きな文字や写真がの余白に流れ込む文字組を再現したい。 float プロパティを持った要素は値にあわせて（左の場合は左 右の場合は右に）要素が移動して後続でかかれたテキストや要素が隙間にまわりこむ。

フロートモードになった要素は内容の幅をとるようになる。レイアウトを変化させたくなければwidthプロパティで幅を入れることを忘れずに。

floatの主な値
  • 値：left [左に流れます。]
  • 値：right　[右に流れます。]
  • 値：none [フロートしません（通常）] 

    /*クラス img要素は左に流れる*/
  img {
  float:left;
}
/*クラス .floatRight は右に流れる*/
.floatRight{
  float:right;
}
float

clearの役割

clearプロパティは上流で設定されたフロートプロパティがあるとき、後続する要素が回り込もうとする動きを通常の表示にもどす。

clearの主な値
  • 値：left [左に流れるのをやる。]
  • 値：right [右に流れるのをやめる。]
  • 値：both [左右に流れるのをやめる。]

/*クラス h1要素は左に流れるの終わり*/
  h1 {
  clear:left;
}
/*クラス .clearRight は右に流れるのやめる*/
.clearRight{
  clear:right;
}
clear

clearプロパティが有効な要素

clearプロパティはブロック要素にしか設定できません。フロートしている文脈が終了し次の文脈は普通の表示にしたいときに設定します。

margin-topが出ない。

clearプロパティをもった要素はmargin-topプロパティがありますが余白が無効になります。上流の文脈がフロートモードになっているからです。

このように　画像や　大きな文字の左右の隙間に文字が流れ込むような文字組をしたい場合floatプロパティをつかいます。

floarの応用

floatが入った要素は幅がコンテンツの幅を自動でとるようになる。文字が一文字の場合がその文字の部分しかはばがひろがらないし。親要素や祖先要素の幅以上の大きさ場合ははみ出して並んだり　祖先の幅を超えるまでは自動改行して並んだりすることがない。

HTML

<ul>
<li class="floatLeft">左</li>
<li class="floatRight">右</li>
<li class="clear">クリア</li>
</ul>

CSS

/*左にフロートだよ*/
.floatLeft{
float:left;
}
/*右にフロートだよ*/
.floatRight{
float:right;
}
/*フロートやめんだよ*/
.clear{
clear:both;
}

横並びをさせたいときは幅を設定する。

ちょうど半分の幅でカラムを作りたい場合

HTML

<div>
<div class="floatLeft">左</div>
<div class="floatRight">右</div>
<div class="clear">クリア</div>
</div>

CSS



/*余白をコンテンツエリアに計上する。*/
div{
  boxsizing:border-box;
}
/*左にフロートだよ*/
.floatLeft{
float:left;
width:50%;
}
/*右にフロートだよ*/
.floatRight{
float:right;
width:50%;
}
/*フロートクリアー*/
.clear{
clear:both;
}

clearの応用

clearfix

クリアする要素が無いとき クリアのする要素を疑似要素で挿入する。

.clearfix:after{
content:"";
display:block;
clear:both;
}

疑似要素セレクター :before:after

要素の内容の最初 最後にテキストなどの要素を挿入する。 HTML要素ではなく見栄えのためだけに挿入するため機械的な意味や翻訳に影響しない。

.quote:before{
content:"「";
}
.quote:after{
content:"」";
}

結論

  • 横並びになる要素にfloatプロパティ
  • 横並びになる要素に横幅のプロパティwidthを設定
  • 横並びになる要素に横幅のプロパティbox-sizing:border-box;を入れておくとと計算が楽
  • フロートを終わらせたい要素にclearプロパティ
  • フロートを終わらせたい要素にがないときclearFixのテクニックをつかいましょう。
