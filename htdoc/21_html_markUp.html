HTMLファイルのファイル名の付け方 - HTML

category: HTML

ここではHTMLファイルの名前の付け方のルールについて解説します。
HTMLファイルの名前のつけかたにはルールがあります。

  • 英数半角でつけること
  • 記号は「-」「_」しかつかえない。
  • 拡張子は「.html」「.htm」であること
  • できればファイル名は内容が類推しやすいこと。
  • 「index.html」はフォルダのトップファイルになる。

使える記号は「-」ハイフン「_」アンダーバーのみ。

OSやサーバーアプリケーションが扱う記号がおおいため。記号として使える文字は少ない。 半角の「-」「_」をうまく使おう。

OKの例

  • sample.html
  • sample_2019.html
  • 2020-tokyo.html

これはユニークになります

同じsampleという文字列で意味的には同じですが別のファイルとして扱われます。見間違えるので注意が必要です。

  • sample.html
  • Sample.html
  • SAMPLE.html

大文字小文字を判別します。

「index」ってつけるとトップぺージになるの？

サーバーの設定で・・・・ index（目録）はとくに指定がなければフォルダをリクエストされた場合 index.（任意の拡張子）優先されて表示されます。 これはセキュリティ的にフォルダの中身をリストアップされないようにサーバー会社が配慮して設定しています。 詳しい設定はサーバー会社の設定画面を読んでください。

トップページは必ず 「index.」になります。

サーバーの公開フォルダー

index.htmlが公開フォルダーの直下にあった場合。

  • http:// example.com
  • http:// example.com/
  • http:// example.com/index.html

クライアントリクエストが上記のURLをたたいてもindex.htmlが表示される。

注意

フォルダの最初に表示されるファイルがindex.htmlですのでフォルダ構成が多いwebサイトはフォルダ毎にいindex.htmlファイルを作ることになります。

編集中どのフォルダのindexファイルを判別しづらくなりますので注意が必要です。
