$の衝突(コンフリクト)

$は便利な変数でphpでも予約文字でつかわれjQuery以外のライブラリーでもグローバル変数として定義されています。もしそのサイトに＄変数が定義された別のjavascriptのファイルが読み込まれていた場合、$を上書きされjQueryが動かないまたはscript自体が動かない場合があります。

基本の記述は2017年でもこちらのようですが、サイトの事情でそうもいかない場合などは下記のような方法や本サイトなどにも＄のコンフリクトを起こさない方法があります。
