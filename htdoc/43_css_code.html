Font-size

font-sizeは文字の大きさを指定します。指定されているfont-familyによって同じ指定をしても大きさが異なる場合があります。

サイズ指定の値は単位が必要です。

基準のフォントサイズ

無指定の場合はブラウザの設定の文字の大きさが標準の文字の大きさになります。ほとんどブラウザの文字の大きさは16pxが標準で「中」と表記されています。

制作するブラウザは文字の設定を「中」にしておく。

視覚弱者は文字をよみやすいように大に設定することを想定してデザインすることも大切。

値

  • xx-small, x-small, small, medium, large, x-large, xx-large

    ユーザーの既定のフォントサイズ (つまり medium) を基準とした絶対的なサイズのキーワードです。

  • larger, smaller

    相対的なサイズのキーワードです。フォントは親要素のフォントサイズから相対的に、上記の絶対的サイズのキーワードで使われている倍率におよそ沿う形で拡大または縮小されます。

  • 長さ

    正の [length] の値。 (em や ex などの) フォント相対単位のほとんどは、親要素のフォントサイズに対する相対値です。

    よくつかわれる距離の単位

    css 長さの単位

    length

    • rem

      ルートイーエム　html要素のフォントサイズを基準に設定する。標準フォントサイズが16pxの場合1remは16pxのフォントサイズをしてすることになります。0.5remは8pxです。

      [求めるフォントサイズ] / [標準フォントサイズ] = [答え]rem　

    • em

      イーエム　親要素のフォントサイズを基準に設定する。親のフォントサイズはその親を継承しているので。祖先要素でフォントサイズを指定している要素を探すとよい。親フォントサイズが30pxの場合1emは30pxのフォントサイズをしてすることになります。0.5emは15pxです。

      [親フォントサイズ] / [求めるフォントサイズ]= [答え]em　

    • px

      直接指定できるため書きやすい指定

      ピクセル　解像度素子数でフォントサイズを指定する。デザインカンプから確認する場合はまずpｘで確認する。

      一部のブラウザは文字拡大や文字縮小に対応しない恐れがあるためpxでの指定は避ける傾向にある。

  • percentage

    正の [%] 値で、親要素のフォントサイズからの相対値です。

引用： font-size

記述例


/* taipo 文字「中」の場合*/
html{
    font-size:100%;/*16px 実質書かなくてよい*/
}
body{
    font-size: .875rem;/*14px*/
}

h1 {
    font-size: 3rem;/*48px*/
}

h2 {
    font-size: 2rem;/* 32px */
}

基準フォントを16px 通常表示するフォントを14pxに設定する。


/* taipo 文字「中」の場合*/
html{
    font-size:62.5%;/*10px*/
}
body{
    font-size: 1.4em;/*14px*/
}

h1 {
    font-size: 4.8rem;/*48px*/
}

h2 {
    font-size: 3.2rem;/* 32px */
}

基準フォントを10px 通常表示するフォントを14pxに設定する。remで設定するとき10pxを基準にするので暗算での指定が楽。
