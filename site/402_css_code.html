Element「エレメント」タブ

Element（要素）タブのはDOMの構成上の要素の構造を確認できます。

要素の内容は△で格納されています。△をクリックすることで展開して内容子孫要素を確認できます。

要素をホバーするとウィンドウで該当の要素が青く色づけされて何処にレイアウトされているかわかります。

左上の□のマークをクリックしてウィンドウでの要素選択モードをONにするとElement（要素）タブではなくウィンドウを選んで要素を確認できます。

  • 1 Element「エレメント」タブ
  • 2 window上の要素を選ぶ
  • 3 styleタブ

styleタブ

styleタブは Elementタブにくっついている、スタイルを確認するための窓です。 上から順番にCSSが該当要素に適応されているのがわかります。　スクロールの最後にボックスモデルの画像があります。ここで要素の余白、幅、高さが確認できます。
