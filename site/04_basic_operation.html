基本的なキーボードショートカット - コンピューターの基本操作

category: コンピューターの基本操作

ショートカットを使いこなすことは作業の短縮の一翼を担います。

マウスオペレーションは便利ですが、アクションに時間がかかります。
キーボードオペレーションは同じ時間でも多くの工数をこなせます。
3秒かかっていたことが1秒で終わる。たった2秒の短縮ですが1年間の仕事時間を加算すればかなりの時間をかせげます。
その分　ぼーっとしていてもこなした仕事量はかわりません。サッサとやってサッサとおうちにかえりましょう。

よく使うソフトはキーボードショートカットを覚える価値があります。

※キーボードショートカットを使ってさっさと帰ろう。

キーボードは10本の手の指を使います。

体を最大限利用しましょう。

もしなんらかの障害などで10本の指をつかえなかったらごめんなさい。でも使える方は最大限つかいましょう。

個人的感想ですが、作業の遅いひと。学習の遅いひとに共通に感じる点は機器にさわりなれていないことです。
意識して右手にはマウスと左手にはキーボードを使うようにしましょう。 ショードカットは両手をつかってはじめて動くものが多いです。特にイラレとフォトショは両手での作業が強いられますので機能を十分に生かせた動きができていないのなら片手での操作はまだ早いです。

windows と macの修飾キーの違い

commandとOptionのキーの位置がちがうもののなれればほとんど同じです。
初心者はMacはcommandキー　windowsはCtrlキーが同じ役割と覚えておくだけでいいです。
このサイトでは　Macのcommandキーを Ctrlと表記するとが多いです。

ちなみにMacにはcontrlキーと呼ばれる別のキーがあります。

日本語jisキーボード 修飾キー

Key mac windows
コマンド command Ctrl
オルト option/Alt Alt
コントロール cotrol 無し
シフト shift shift

アプリケーションのほとんどにはショートカットキーがあります。 下記のショートカットキーはおおよそ共通のショートカットとして見られるキー配置になります。

使っているアプリケーションのヘルプメニューや　メニューバーのコマンド名の左にショートカットキーの案内があるはずです。ここをみながら使って覚えます。

ファイル操作のショートカットキー

ショートカットキーは使わないと指がうごきません。　使って覚えこみましょう。

機能 ショートカットキー
保存/新規保存 Ctrl+s
命名保存/別名保存 Ctrl+shift+s
開く Ctrl+o
アプリ終了 Ctrl+q

特に保存から覚えること。作業したら保存の癖は身に着けるべきスキル

編集メニューのショートカット

機能 ショートカットキー
コピー Ctrl+c
貼り付け Ctrl+v
カット/移動 Ctrl+x
検索 Ctrl+f
置き換え Ctrl+h
編集をやり直し Ctrl+ｚ
編集のやり直しのやり直し Ctrl+y
編集のやり直しのやり直し Ctrl+shift+z

選択メニューでのショートカット

機能 ショートカットキー
すべてを選択 Ctrl+a
行を選択 Ctrl+L

表示メニューでのショートカット

機能 ショートカットキー
拡大 Ctrl++
縮小 Ctrl+-
表示100％ Ctrl+0

触れるしかない　つかうしかない

一つ一つ使うことからはじめよう

最初はショートカットを忘れてついついマウスでのオペレーションを使いがちになります、よく使うアプリの機能はショートカットキーを覚えてもっとアプリを楽しんでください。
