CSS vertical-align プロパティ - css

category:css

行の表示位置　垂直方向の位置を変更できるプロパティです。行(inline)内の指定するプロパティと表のセルボックス内の文字の表示位置を制御する2種類の値があります。

目次

  • vertical-align

vertical-align

CSS の vertical-align プロパティは、インラインボックスまたは表セルボックスの垂直方向の配置を設定します。

値

  • baseline

    ほとんどの要素の規定値

    要素のベースラインを親要素のベースラインに揃えます。 <textarea$gt; のような、一部の 置換要素 のベースラインは HTML 仕様で未定義であり、このため、このキーワードの挙動はブラウザにより異なるかもしれません。

  • sub

    要素のベースラインを親要素の subscript-baseline に揃えます。
    sub要素の規定値になっていることが多いです。

  • super

    要素のベースラインを親要素の superscript-baseline に揃えます。
    sup要素の規定値のはず

  • text-top

    要素の上端を親要素のフォントの上端に揃えます。

  • text-bottom

    要素の下端を親要素のフォントの下端に揃えます。

  • middle

    要素の下端を親要素のフォントの下端に揃えます。

  • length

    要素の下端を親要素のフォントの下端に揃えます。

  • percentage

    値と似ていますが、line-height プロパティに対するパーセンテージで指定します。負の値を使えます。

行との相対値

親要素ではなく、行全体に対しての縦方向の整列方法を表します。

  • top

    素と子孫要素の上端を行全体の上端に揃えます。

  • bottom

    要素と子孫要素の下端を行全体の下端に揃えます。
    ベースラインを持たない要素では、代わりにマージン境界の下端が使われます

基準になる親テキストにはtext-decolationプロパティunderlineとlineheightを2にしています。旗の絵文字がvartical-alignの対象文字で3emの大きさで表示しています。

表セル内での値

表の枠（セル）内ではcontentsエリアの描画位置（垂直位置）の指定に使えます。プレビューはそれぞれ「テストテキスト」の描画位置で確認してください。

プレビューはTD要素にのみ設定しています。

  • baseline (および sub, super, text-top, text-bottom, 「length」, 「percentage」)

    セルのベースラインを、行内でベースライン揃えにされた他のすべてのセルのベースラインに揃えます。

  • top

    セルの上端のパディング境界を行の上端に揃えます

  • middle

    セルのパディングボックスが行の中央になるようにします。

  • bottom

    セルの下端のパディング境界を行の下端に揃えます。

プレビューはTD要素にのみ設定しています。
