CSS font プロパティ 1 - css

category:css

font系プロパティは文字の見た目を制御するプロパティです。

font系プロパティのほとんどは祖先要素より子孫要素に継承します(一部のインタラクティブ要素は除く)。そのためある要素に設定されたプロパティはその子孫に上書するプロパティがかかれない限り祖先で設定されたプロパティを継承します。

目次

  • font-family
  • font-size

font-family

font-familyは書体を指定するプロパティです。

複数の値をコンマで区切って、代替フォントを示します。ブラウザーはリストの中で最初のインストール済み、または @font-face @-規則を使用してダウンロード可能なフォントを選択します。

注意点としてはクライアントのシステムにあるフォントしかユーザー設定できません。しかしサイトの表示に時間がかかるリスクは少ないです。

一括指定プロパティの font を使用すると、 font-size やその他のフォント関連プロパティを一度に設定できるのでふつうは便利です。

指定されたフォントが利用可能であるという保証はないので、 font-family リストの中に、総称ファミリを少なくとも 1 つ、常に追加しておくべきです。総称ファミリを使うことで、必要ならば、ブラウザーで受け付け可能な代替フォントを選択させることができます。

とりあえずゴシック系

  html{
    font-family: sans-serif;
  }

値

  • <family-name>

    フォントファミリの名称。例えば、 "Times" や "Helvetica" はフォントファミリです。空白を含むフォントファミリ名は、引用符で囲む必要があります。

  • <generic-name>

    総称フォントファミリは代替メカニズムです。このメカニズムによって、指定されたフォントがどれも利用できなかった場合、スタイルシート製作者の意図を多少なりとも保つことができます。総称ファミリ名はキーワードであり、引用符で囲んではいけません。総称フォントファミリは、フォントファミリ名リストの最終選択肢である必要があります。以下のキーワードが定義されています。

  • serif

    終わりが止めや払い、または実際の活字のセリフの形をした角を持つ字形です。 例: Lucida Bright, Lucida Fax, Palatino, "Palatino Linotype", Palladio, "URW Palladio", serif

    解像度の低いwebでは漢字の明朝体がつぶれますので　小さな文字の読み文字にはお勧めできません。 和風なデザインや格式など感じるイメージなので見出しなどに使うことがあります。

  • sans-serif

    平たい角の端を持つ字形です。 例: "Open Sans", "Fira Sans", "Lucida Sans", "Lucida Sans Unicode", "Trebuchet MS", "Liberation Sans", "Nimbus Sans L", sans-serif

    • 解像度の低いwebではゴシックが定番です。webの読み文字 見出しともよくつかわれます。

  • monospace

    すべての字が同じ幅を持つ字形です。 例: "Fira Mono", "DejaVu Sans Mono", Menlo, Consolas, "Liberation Mono", Monaco, "Lucida Console", monospace

    プログラム用のテキスト

引用： font-family

よく見るフォントファミリー定番指定

システムフォントを駆使する方法

MAC OSとwindowsの両方で指定できるゴシック系フォント


/*Yu Gothic*/
html{
   font-family: "游ゴシック体", YuGothic, "游ゴシック Medium", "Yu Gothic Medium", "游ゴシック", "Yu Gothic", sans-serif;
}

MAC OSとwindowsでは同じアウトラインだが重さが違う。

Chrome の描画がやや薄く表現されるためillustratorの描画と差異がでるのでタイポグラフィーにこだわるサイトでは問題になる。

webフォント

Webフォントを駆使する方法

HTMLヘッダーにフォントデータを読み込む

<link href="https://fonts.googleapis.com/css?family=Noto+Sans+JP" rel="stylesheet">

要素に設定する


/*noto Gothic*/
html{
  font-family: "Noto Sans JP", sans-serif;
}

書体はデザインの情緒感を表現する大切な要素です
しかし、現状は 多くの会社が様々なデバイスを販売しそれぞれのデバイスでユーザーが閲覧している限りべてのデバイスで再現することは不可能に近いです。

サイトのデザインの方向性に合わせた書体をえらびましょう。ブランドが重要なサイトなどはフォントを犠牲にできないでしょう。逆に公共性に高いサイトはどんなフォントが重要ではなく伝わるかどうかが大切です。

ローマ字と日本語フォント

ローマ字を先日本語をあとに指定する。

半角英字を日本語フォントにあるフォント以外にしていたい場合はローマ字フォントを先に指定する。


/*Yu Gothic*/
@import url('https://fonts.googleapis.com/css?family=Open+Sans&display=swap');
html{
   font-family: "Open Sans" "游ゴシック体", YuGothic, "游ゴシック Medium", "Yu Gothic Medium", "游ゴシック", "Yu Gothic", sans-serif;
}

ローマ字にOpen Sansが先にセットされ日本語の場合はフォントがないので游ゴシックが設定される。
